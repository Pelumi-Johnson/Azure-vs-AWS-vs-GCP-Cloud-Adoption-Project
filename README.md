## üìå Project Overview
This project analyzes how cloud adoption can support Don & Associates a 200 employee consulting firm headquartered in New Jersey during its planned expansion across the East Coast. The company faces challenges with on-premises servers, limited scalability, slow collaboration tools, and high maintenance workloads. This report evaluates the top three cloud providers and offers a migration recommendation tailored to business growth.

# üåê Cloud Provider Analysis

---

## ‚òÅÔ∏è Amazon Web Services (AWS)
**Service Models:** IaaS, PaaS, limited SaaS  
**Core Tools:**  
- Compute: EC2  
- Storage: S3  
- Databases: RDS, DynamoDB  
- Developer Tools: CodeBuild, CodePipeline  
- Analytics: Redshift, QuickSight  
- Networking: VPC, CloudFront  

**Strengths:**  
- Largest global cloud infrastructure  
- Highly flexible & technical  
- Pay-as-you-go pricing (complex but customizable)

---

## ‚ö° Microsoft Azure (Recommended)
**Service Models:** IaaS, PaaS, SaaS  
**Core Tools:**  
- Compute: Azure VMs  
- Storage: Blob Storage  
- Databases: Azure SQL, Cosmos DB  
- Dev Tools: Visual Studio, Azure DevOps  
- Analytics: Azure Monitor, Synapse  
- Networking: Azure Virtual Network  

**Strengths:**  
- Best integration with Microsoft 365  
- Strong security & compliance tools  
- Hybrid cloud + off-premises support  
- Smooth migration from Windows-based environments  
- Excellent enterprise support & cost management  

---

## üîµ Google Cloud Platform (GCP)
**Service Models:** IaaS & PaaS  
**Core Tools:**  
- Compute: Compute Engine  
- Storage: Cloud Storage  
- Databases: Cloud SQL, Bigtable  
- Dev Tools: Cloud SDK, Firebase  
- Analytics: BigQuery  
- Networking: Cloud Load Balancing  

**Strengths:**  
- Clean UI, developer-friendly  
- Transparent pricing  
- Growing global footprint  

---

# üèÜ Final Recommendation: **Microsoft Azure**
Azure best supports the company's Microsoft-based environment and offers:
- Seamless integration with Microsoft 365  
- Strong hybrid-cloud options for gradual migration  
- High security + compliance standards  
- Scalable storage, compute, and analytics  
- Reliable global infrastructure  
- Transparent cost-management tools  

Azure provides the most balanced, future-ready platform for business expansion.
